# Domain Glossary

## Proto Cook

Prototype host app for reusable future project decisions. Proto Cook is not a product vertical; it is the app shell where platform capabilities, Feature Labs, and Vertical Modules are exercised before extraction or reuse.

## Vertical Module

Domain-specific feature set inside Proto Cook, such as courses, commerce, social, or warehouse. A Vertical Module may include routes, persistence, API procedures, labs, and docs, but it does not rename the host app.

## Feature Lab

Gated/manual surface used to test implementation approaches and UX/DX. A Feature Lab is allowed to expose comparison controls, failure-state toggles, and diagnostic output that would not belong in a default product flow.

## Files Platform

Reusable files/upload/storage/processing/playback packages plus the app adapters that connect them to auth, persistence, queues, telemetry, and deployment infrastructure.
